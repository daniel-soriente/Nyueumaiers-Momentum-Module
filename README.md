Momentum Trading Strategy
================

- Implements Momentum trading algorithm to analyse trades and make decisions on them
- Implemented emphasising component programming (to be added to an exciting system)
- Input CSV file containing trade data (Must follow input from Aus Equities order book format - http://www.cse.unsw.edu.au/~fethir/TradingEngineCourse/Session1/section%201%20links%20and%20topic/Revised%20introOrderbook-1-3.htm)
- Outputs CSV containing decisions on trades
- Run 'NyueumaiersModule.jar' for live demo (Java must be installed)
